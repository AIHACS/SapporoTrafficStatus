# 札幌混雑状況予測ウェブサイト

20260616作成

札幌市内で開催される主要イベントについて、地下鉄・JR・道路の混雑しやすい時間帯や交通規制の目安を確認できるWebアプリです。

イベント名、月、交通手段、影響レベルで絞り込みながら、イベントごとの混雑ピーク、影響を受けやすい駅・エリア、公式サイトへのリンクをまとめて確認できます。

札幌という課題で，札幌市民は「なんか今日電車混んでるなー何かあるのかな？」となることが多い．
そこで，札幌の大きなイベントをまとめていつ混雑するのか？道路はどこが封鎖されるのか？をAIによる予測を立てる目標を立てこのウェブサイトを作成した．

## サイトURL

https://aihacs.github.io/SapporoTrafficStatus/

## 主な機能

- イベント名、駅名、エリア名による検索
- 月別の絞り込み
- 地下鉄・JR・道路など交通手段別の絞り込み
- 交通影響レベル別の絞り込み
- イベントごとの混雑予想時間、交通規制メモ、関連駅情報の表示
- スマートフォン表示に対応したレスポンシブレイアウト

## 掲載しているイベント

- 第35回 YOSAKOI ソーラン祭り
- 北海道神宮例祭（札幌まつり）
- 北海道マラソン 2026
- さっぽろ雪まつり
- 札幌マラソン（第51回）
- さっぽろ夏まつり
- No Maps

## 使い方

サイトにアクセスすると、イベント一覧と詳細情報が表示されます。

検索欄や各フィルターを使うと、確認したいイベントや交通手段に絞って表示できます。イベントカードを選択すると、混雑しやすい時間帯、影響を受けやすい駅、交通規制に関するメモを確認できます。

## ローカルでの確認方法

このアプリは静的なHTML/CSS/JavaScriptで構成されています。ローカル環境では、`index.html`をブラウザで開くことで確認できます。

```text
SapporoTrafficStatus/
├── index.html
├── styles.css
├── app.js
└── assets/
    └── sapporo-flow.png
```

## データについて

掲載している混雑予想や交通規制情報は、イベント情報と主要交通結節点をもとにした目安です。

実際の運行状況、交通規制、開催日時は変更される場合があります。移動前には、各イベント公式サイトや交通機関、道路交通情報などの公式発表を確認してください。
